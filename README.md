# Description
  A static Event website built using **HTML5** and **CSS** , showcasing event details, Blogs.

## Pages OverView

  **Home (index.html)**
    Landing page with event overview and blog.

  **Events (event.html)**
    All Events and the details of each one.

  **Blog (blog.html)**
    All Blogs .

  **login (login.html)**
    Simple login form .

  **Sign up (signup.html)**
    Simple Sign up form.
